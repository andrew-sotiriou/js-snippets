Arrays

MAP

Use MAP when you want to perform the same action on every item in an array

Example:

const numbers = [1, 2, 3, 4, 5];
const squareNumbers = numbers.map(number => { return number * number; });

FILTER

Use FILTER when you only want items that match a certain criteria

Example:

const things = ['desk', 'chair', 5, 'backpack', 3.14, 100];
const onlyNumbers = things.filter(thing => { return typeof thing === 'number'; });
