JQuery AJAX

JQUERY AJAX GET

This is a simple JQuery AJAX Get call.

Example:

$.ajax({
url:"https://restcountries.eu/rest/v2/name/cyprus",
type: "GET",
dataType: 'json'
}).done(function(data){
doSomething(data);
}).fail(function(error){
doSomething(error);
});

JQUERY GET Call

This is a simple JQuery using an updated GET CALL. This is a step up from the lower level called utilized in the above snippet.

Example:

$.get('https://restcountries.eu/rest/v2/name/france', data => {
doSomething(data);
}).fail( error => {
doSomething(error);
